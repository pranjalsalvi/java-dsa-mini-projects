# Java DSA Mini Projects

This repository contains basic Java programs focused on data structures, algorithms, and API handling.

## Programs Included

1. **Nodes at Distance K (Binary Tree)**

   * Uses recursion to print nodes at a given distance from root.

2. **Bitcoin Price Fetcher (API + Number to Words)**

   * Fetches live Bitcoin price using REST API
   * Converts INR value into words (Indian number system)

3. **Orchard Size Detection (Matrix + DFS)**

   * Uses Depth First Search (DFS)
   * Finds connected components in 8 directions

## Tech Used

* Java
* VS Code
* REST API
* DFS & Recursion

## How to Run

```bash
javac FileName.java
java FileName
```
